# Easyly - Koleksi Tools Produktivitas Harian

**Easyly** adalah platform web statis yang menyediakan berbagai macam *tools* praktis untuk mempermudah aktivitas sehari-hari Anda. Proyek ini dibangun dengan framework [Astro](https://astro.build/) untuk memastikan performa yang cepat, ringan, dan ramah pengguna.

##  Live Demo
Kunjungi proyek ini secara langsung di:
 **[https://rm19x.github.io/easyly/](https://rm19x.github.io/easyly/)**

##  Fitur & Tools yang Tersedia
Easyly saat ini dilengkapi dengan berbagai alat bantu, di antaranya:

- **Surat & Dokumen**: Generator draf surat lamaran kerja dan surat perjanjian jual beli.
- **Kalkulator & Estimasi**: Penghitung zakat, kalori makanan, biaya listrik, dan penyusutan aset.
- **Kesehatan & Keamanan**: Panduan P3K, kalender masa subur/menstruasi, dan generator password aman.
- **Produktivitas & Kreatif**: Pengubah kapitalisasi teks, generator QR Code, tes kecepatan mengetik, dan log servis kendaraan.

##  Teknologi yang Digunakan
Proyek ini dikembangkan dengan teknologi modern:
- **Framework**: [Astro](https://astro.build/)
- **Deployment**: [GitHub Pages](https://pages.github.com/)
- **Styling**: CSS murni (Native) untuk kecepatan maksimal.

## 📦 Cara Menjalankan Secara Lokal
Jika Anda ingin mengembangkan atau menjalankan proyek ini di komputer lokal:

1. Pastikan Anda sudah menginstal [Node.js](https://nodejs.org/).
2. Clone repositori ini:
   ```bash
   git clone https://github.com/Rm19x/easyly.git
   cd easyly
   npm install
   npm run dev
   Buka http://localhost:4321 di browser Anda.
   ```
